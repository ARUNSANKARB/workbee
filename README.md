# 🐝 WorkBee – Local Skill Exchange Platform

WorkBee is a full stack MERN web platform that connects customers with local skilled workers such as electricians, plumbers, cooks, maids, and homemade catering providers.

The platform enables users to discover services, book professionals, track booking status, and leave reviews — while providing workers with a digital presence and job opportunities.

---

##  Features

###  Customer

* User authentication (JWT)
* Browse services by category
* View worker profiles
* Book services
* Track booking status
* Review & rating system
* Booking history dashboard

###  Worker

* Worker profile & skill registration
* Document upload for verification
* Accept/reject booking requests
* Availability management
* Earnings dashboard

###  Admin

* Worker verification
* Category management
* User moderation
* Analytics dashboard

##  Installation

### Clone repo

```bash
git clone https://github.com/your-username/workbee.git
```

### Backend setup

```bash
cd backend
npm install
npm run dev
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```
